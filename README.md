# churnwrapper
API wrapper for ChurnOver

# About
This is a simple API wrapper in Python for getting Colorado business data from the ChurnOver project.

Valid search parameters include:

    'agentprincipaladdress1',
    'principalzipcode',
    'entityformdate',
    'principalstate',
    'entitytype',
    'agentlastname',
    'agentprincipalcity',
    'entitystatus',
    'entityid',
    'entityname',
    'agentfirstname',
    'location',
    'agentprincipalcountry',
    'agentprincipalstate',
    'principaladdress1',
    'entitytypeverbatim',
    'agentprincipalzipcode',
    'principalcity',
    'principalcountry',
